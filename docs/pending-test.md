# 待测试
- 画布连线删除只保留右键删除，不再显示线上的小叉号；拖动连接线时增加节点边缘和左右连接点的吸附命中范围，连接图片节点更容易命中；提示词输入框右键菜单改为“粘贴提示词”和“粘贴参考图”，支持直接读取剪贴板文本或图片，浏览器不允许读取时提示使用 Ctrl+V。
- 后台图片分组新增默认画质配置；画布新建配置节点、从文本节点生成配置节点、批量导入生成配置节点时默认使用该分组默认画质；生图面板会直接显示连入的参考图缩略图并按最终请求顺序标记为 `图1`、`图2`，调整顺序会同步影响请求中的参考图数组顺序；删除连接线会同时删除引用关系；画布工具栏支持将全部图片或选中图片打包下载为 zip。

- 新增 `/parse` 解析工作台：后台模型管理支持添加“解析模型”，用户侧 API Key 配置弹窗可选择并填写解析模型密钥；解析工作台左侧输入分享链接或完整分享文本，右侧按视频、图片、音频三列展示在线预览，并支持复制链接、下载媒体，视频下载文件名固定为 `.mp4`。
- 修正解析工作台媒体归类和中文标签：`music url` 归入音频，`music avatar` 作为音频封面显示在音频列，作者头像、视频封面、音频封面、视频、音频、图片等条目统一显示中文名称，普通 `url/link` 不再作为不可播放视频展示。
- 后台模型管理改为图片分组配置：图片分组可填写请求地址、`512`/`1k`/`2k`/`4k` 对应模型 ID 和支持比例；用户 API Key 配置弹窗平铺显示所有启用分组/模型，只有填写了 Key 的图片分组才会出现在画布生图模型下拉中，切换分组后清晰度和比例选项实时按该分组刷新，实际请求按当前清晰度发送对应模型 ID。
- 新增 `/prompt` 提示词工作台：后台模型管理支持添加“提示词模型”并配置后台专用 API Key；用户无需配置密钥，可免费上传图片提取中文生图提示词，结果支持编辑和复制。
- 图片分组新增“参考图数量”配置；画布提示词输入面板新增加号上传参考图，输入框粘贴图片会直接转成参考图并连接到当前生成节点，上传/粘贴数量按当前图片分组限制拦截。
- 画布添加参考图后，参考图节点默认以较小显示尺寸进入画布，不压缩原图；提示词输入框右键菜单独立于图片节点右键菜单，支持添加参考图和粘贴提示词文本。
- 画布右侧助手开放生图功能，移除对话模式，只保留图片生成、参考图、模型和图像设置，并沿用画布任务状态轮询链路；提示词输入框右键菜单改为挂载到页面根层级，避免在画布节点内定位失效。
- 画布和视频工作台会记住用户上次选择的模型，下次打开默认使用该模型；清晰度、比例、生成数量等参数不跟随这次选择写入全局默认。
- 画布图片节点右键菜单新增“复制图片”，主图和批量图片节点都可以直接复制图片到系统剪贴板。

- 新增后端模型管理接口：公开 `/api/models` 返回已启用模型，后台 `/api/admin/models` 支持模型列表、新增、编辑和删除；API Key 配置弹窗和知梦后台模型管理页不再因接口缺失提示加载模型列表失败。
- 前端不再提供用户登录入口，`/login` 会直接回到首页；用户侧统一通过配置弹窗选择模型并填写 API Key，只有知梦后台 `/zhimengapi/login` 保留管理员登录。
- 全站 Logo 替换为新的 3D M SVG，并改为直接显示 SVG 图片，保留原始渐变颜色。
- 画布生图模型下拉只显示后台配置的模型名称，不再根据模型名展示 OpenAI、Claude 等模型图标；视频工作台模型区域移除“1000 积分/次”文案。
- 后台模型管理新增“调用模型 ID”，前端下拉显示后台配置的中文名称，实际请求上游时发送该模型 ID；例如显示名称可为“ChatGPT生图中转站”，调用模型 ID 填 `gpt-image-2`。
- 知梦 API Key 配置弹窗会通过当前 `IMAGE_API_BASE_URL` 对应站点的 `/v1/models` 检测 Key 是否属于本站点；配置弹窗支持重新保存 API Key，并可手动通过 `/api/usage/token/` 按间隔刷新，以 USD 和绿色进度条显示 1k、2k、4k 各档位余额。
- 生图请求会按当前图像设置里的 1k、2k、4k 档位选择对应 API Key，不再因为全局缓存的 `apiKey` 回退到 1k。
- 知梦 API Key 余额会在前端本地配置恢复后按 `NEXT_PUBLIC_TOKEN_USAGE_AUTO_REFRESH_INTERVAL_MS` 自动刷新，默认每 10 分钟一次；多个 Key 的余额请求会按 `NEXT_PUBLIC_TOKEN_USAGE_REQUEST_DELAY_MS` 间隔发送，默认间隔 3 秒；刷新页面时如果未超过自动刷新周期，会直接复用本地缓存余额，不会立刻请求额度接口。
- 画布右上角余额入口固定显示“用户余额”和状态点，不再直接显示单个 Key 余额；悬浮时展示 1k、2k、4k 三档余额进度条和 `$x / $y`，并可直接点击“刷新额度”；所有已配置 Key 余额均不少于一半时状态点显示绿色。
- 画布项目导出改为下载 `.zip` 压缩包，包内包含 `projects.json` 和当前画布引用到的本地图片、视频文件，避免只导出 JSON 时丢失媒体内容。
- 画布库支持多选后一键导出多个画布项目，导出的压缩包可一次恢复多个项目。
- 画布项目导入改为读取新版 `.zip` 压缩包，会先按 `projects.json` 中的文件映射恢复图片、视频到本地存储，再插入画布项目，导入成功后仍停留在画布库。
- 修复删除画布图片节点或清空画布后撤销时，节点信息恢复但本地图片数据已被清理导致图片丢失的问题。
- “我的素材”类型筛选区右侧新增文本样式的导出素材和导入素材入口，可将全部素材导出为包含 `assets.json` 与图片、视频文件的压缩包，并从压缩包恢复素材。
- 未配置 API Key 时，用户侧顶部和画布右上角不再显示登录入口；Key 无效时直接提示重新配置并打开 API Key 配置弹窗。
- 生图工作台的图片参数区复用画布里的紧凑版图像设置面板，尺寸、质量、生成张数的交互保持一致；工作台仍保留独立的模型选择。
- 生图工作台新增生成记录配置持久化：每次生成会保存提示词、参考图、模型、质量、尺寸和张数，结果图写入本地图片存储后记录只保存 `storageKey`；点击历史记录会回填本次生成配置并预览结果。
- 视频设置抽成画布和视频创作台共用的紧凑面板，清晰度、尺寸、秒数按固定网格选择并支持手动输入；尺寸选择 `auto` 时请求不传 `size`。
- 修复画布和生图工作台选择图片尺寸后，请求图片生成/编辑接口未携带 `size` 参数的问题；`auto` 不传，其余比例或像素尺寸会随请求发送。
- 修复生图工作台和画布生图请求中 `quality` 参数可能传入上游不支持值导致 400 的问题；请求前会归一化质量枚举，`auto` 或异常值不再发送给上游。
- 管理后台新增/编辑渠道时，渠道可用模型支持通过弹窗按“新获取、已有”分组选择，并可在弹窗内手动增加模型或拉取模型列表后再写回表单。
- 管理后台编辑渠道时，API Key 留空不再触发必填校验，表示沿用已保存的密钥；新增渠道仍要求填写 API Key。
- 管理后台公开配置里的系统可用模型候选项改为由已启用渠道中选择的模型合并去重生成，最终开放哪些模型仍由公开配置里手动勾选。
- 视频生成请求参数对齐 `grok-imagine-video` 接口：使用 `resolution_name`、`preset=normal`、`input_reference[]`，支持清晰度、尺寸、秒数快捷选择和手动输入，并支持最多 7 张参考图。
- 画布视频设置浮层改为挂载到页面根层级并使用自建浮层交互，避免被节点悬浮工具栏遮挡或点击面板内容时关闭。
- 画布生成配置节点的生图参数改为复用图像设置浮层，支持在同一个入口里调整质量、尺寸和生成张数。
- 视频清晰度输入框改为只输入数字，提交请求时再拼接 `p` 单位。
- 视频生成前端会识别后端 `{ code, msg }` 错误响应，创建失败不再继续轮询 `undefined`。
- 新增 `/video` 视频创作台页面，参考生图工作台布局，支持提示词、参考图、视频参数、生成结果、保存素材、下载和本地生成记录；清晰度和秒数均支持常用值选择与手动输入，生成记录只保存媒体 `storageKey` 并可回填本次提示词、参考图和参数。
- 视频创作台生成前会把模型、尺寸、秒数、清晰度归一化为视频接口支持的参数，并展示后端返回的错误信息，避免页面侧残留的生图参数影响视频请求。
